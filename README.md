# Website Monitoring Jaringan Sederhana

Website statis untuk menampilkan data perangkat jaringan, status online/offline, ping log, pencarian device, kategori perangkat, grafik dashboard, dan laporan koneksi.

## Cara membuka

Buka file `index.html` langsung di browser.

## Fitur

- Dashboard jumlah device online/offline dan rata-rata ping.
- Daftar perangkat dengan pencarian nama, IP, lokasi, dan MAC.
- Filter kategori dan status.
- Detail device berisi IP, MAC, lokasi, status, packet loss, dan log ping.
- Ping test simulasi untuk device terdaftar atau IP manual.
- Report koneksi dengan export CSV dan cetak.
- Data tersimpan otomatis di `localStorage`.
